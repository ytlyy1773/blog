## Blog
基于vitepress开发的个人博客

## 核心亮点

* 提交代码到一个代码仓库自动同步其他代码仓库
* 自动打包发布github pages网站
* 运行linux服务器脚本执行拉取代码、打包部署操作

```yarn
yarn dev
```

```pnpm

```
